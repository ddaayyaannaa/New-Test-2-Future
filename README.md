FizzBuzz.

To run the FizzBuzz script, clone this repository, navigate to the fizzbuzz folder and execute:

```bash
node index.js
```

Make sure to have node installed.

## Project structure
```bash
src/
  ├── services/
  │   └──fizzBuzzService.js
  ├── utils/
  │   └── fileHandler.js
  │   └── primeChecker.js
  └── index.js
```
